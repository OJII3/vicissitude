# Minecraft 基礎知識

## 昼夜サイクル

- 1日 = 20分（リアルタイム）
- 昼間（0〜12000 tick）: 安全に屋外活動可能
- 夕方（12000〜13000 tick）: 帰還準備
- 夜間（13000〜23000 tick）: 敵mobスポーン、屋内退避推奨
- 夜明け（23000〜24000 tick）: 屋外活動再開可能

## 危険mob

- **ゾンビ**: 夜間スポーン。遅い。日光で燃える。
- **スケルトン**: 夜間スポーン。弓矢で遠距離攻撃。日光で燃える。
- **クリーパー**: 夜間スポーン。接近すると爆発。日光で燃えない。
- **エンダーマン**: 夜間スポーン。目を合わせると敵対。
- **クモ**: 夜間敵対。昼間は中立。

## 体力・空腹

- 体力: 最大20（ハート10個）。6以下で危険。
- 空腹度: 最大20。0になると体力が減少。
- 食料は常に携帯する。

## 基本行動パターン

### 序盤（装備なし）

1. 木を伐採 → 作業台作成
2. 木のツール作成 → 石のツール作成
3. 仮拠点作成（穴掘りでもOK）
4. 食料確保

### 中盤（石〜鉄装備）

1. 鉄鉱石採掘 → 鉄装備作成
2. 本格的な拠点建設
3. 農場作成
4. 探索範囲拡大

### 安全基準

- 体力10以下: 食料摂取・安全な場所へ退避
- 体力6以下: 即座に退避、戦闘回避
- 夜間: 拠点に帰還、または穴を掘って待機
- 未知の洞窟: たいまつで照らしながら慎重に進む
