# Minecraft 基礎知識

## プレイヤーの基本情報

- プレイヤーの高さ: **2 ブロック**（足元 1 ブロック + 頭 1 ブロック）
- 幅: 0.6 ブロック（約 1 ブロック幅の通路を通過可能）
- ジャンプ高さ: 1.25 ブロック（1 ブロック段差は飛び越えられる）
- シェルター構築時は最低 2 ブロックの天井高さが必要

## 昼夜サイクル

- 1日 = 20分（リアルタイム）= 24000 tick
- 朝（0〜6000 tick）: 安全に屋外活動可能
- 昼（6000〜12000 tick）: 安全に屋外活動可能
- 夕方（12000〜13000 tick）: **帰還準備を開始**。拠点方向へ移動
- 夜間（13000〜23000 tick）: 敵mobスポーン。即 sleep_in_bed、ベッドなし→ find_shelter
- 夜明け（23000〜24000 tick）: 屋外活動再開可能

### 昼夜別行動ルール

- 夕方になったら現在のジョブを中断し帰還を最優先にする
- 夜間に屋外にいる場合: sleep_in_bed → 失敗時 find_shelter
- ベッドも避難所もない場合: 足元を3ブロック掘って待機（緊急シェルター）

## 危険 mob と対処法

### 即逃走（flee_from_entity 使用）

- **クリーパー**: 16ブロック以内で逃走。接近すると爆発。日光で消えない。絶対に近づかない
- **ウォーデン**: 見つけたら即逃走。極めて高い攻撃力。戦闘不可

### 距離を取る（16ブロック以上離れる）

- **スケルトン**: 弓矢で遠距離攻撃。遮蔽物に隠れるか逃走。日光で燃える
- **ウィッチ**: ポーション攻撃。距離を取って回避
- **ガスト**: ネザーで出現。火の玉攻撃

### 接近戦可能（装備があれば）

- **ゾンビ**: 遅い。日光で燃える。装備があれば対処可能
- **クモ**: 夜間のみ敵対。昼間は中立。壁を登る
- **エンダーマン**: 目を合わせなければ安全。敵対時は水辺へ逃げる

### 距離閾値

- 8ブロック以内: **危険** — 即座に対応（P0）
- 16ブロック以内（クリーパー、ウォーデン）: 即逃走（P0）
- 16〜32ブロック: 警戒。接近を監視

## 体力・空腹

### 基本情報

- 体力: 最大20（ハート10個）
- 空腹度: 最大20
  - **18以上**: 体力が自然回復する
  - **0**: 体力が減少し始める

### Reactive Layer の自動処理

体力低下時の食事と hostile mob からの逃走は **Reactive Layer が自動処理** します。
閾値に基づく手動判断は不要です。Reactive Layer が処理できなかった場合のみ、イベント通知経由で対応してください。

### 食料の優先度（回復量順）

1. golden_apple（+4 吸収ハート付き）
2. cooked_beef / cooked_porkchop（+8）
3. cooked_mutton / cooked_salmon / cooked_chicken / golden_carrot（+6）
4. bread / cooked_cod / baked_potato / cooked_rabbit（+5）
5. apple（+4）
6. carrot（+3）/ melon_slice（+2）

- 常に食料を6個以上携帯する
- golden_apple は緊急時専用

## 基本行動パターン

### 序盤（装備なし）

1. 木を伐採 → 作業台作成
2. 木のツール作成 → 石のツール作成
3. 仮拠点作成（穴掘りでもOK）
4. 食料確保

### 中盤（石〜鉄装備）

1. 鉄鉱石採掘 → 鉄装備作成
2. 本格的な拠点建設
3. 農場作成
4. 探索範囲拡大

## バイオーム別注意点

- **砂漠**: ハスク（空腹効果付きゾンビ）出現。水・食料を多めに携帯
- **海洋**: ドラウンド（水中ゾンビ）注意。深海は避ける
- **ネザー**: ガスト・ブレイズ・マグマキューブ出現。溶岩に注意。ベッド使用不可（爆発）
- **洞窟**: たいまつ必須。クリーパーの不意打ちに注意。高低差に注意
- **雪原**: ストレイ（鈍足矢スケルトン）出現。視界が悪い場合あり

## 安全基準まとめ

- **Reactive Layer が自動処理**: 体力低下時の食事、hostile mob からの逃走、死亡後リスポーン
- 夜間: 拠点に帰還、sleep_in_bed、または穴を掘って待機
- 未知の洞窟: たいまつで照らしながら慎重に進む
- 常に食料と松明を携帯する
- クリーパー・ウォーデンへの接近攻撃は絶対禁止
